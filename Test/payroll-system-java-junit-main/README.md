# Payroll system in java and junit

Logged in as admin you can:
1. View Account 
2. Create User 
3. See Users 
4. Requests 
5. Pay salary 
6. Log out 
7. Turn off program 
8. See Menu again

Logged in as user you can:
1. View Account 
2. Request change of salary
3. Request change of role
4. Delete My Account
5.Log out
6. Turn off program
7. Back to Menu
